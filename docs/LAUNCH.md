### Description:
```
GitGuard is an open-source git security scanner that detects secrets, vulnerabilities, and automatically fixes your code.

Key features:
- 100+ secret patterns (AWS, Azure, GCP, GitHub, Stripe, OpenAI, and 50+ services)
- 49 vulnerability patterns across 12 languages
- Auto-fix for 11 common security issues
- AI-powered code review with 18 providers (OpenAI, Anthropic, Ollama, etc.)
- Git history scanning to find old leaked secrets
- SARIF output for GitHub Security tab
- SBOM generation
- Web dashboard
- VS Code extension

Install: pip install gitguard
GitHub: https://github.com/gitguard/gitguard

What makes it different from Gitleaks/TruffleHog:
- It fixes your code automatically
- It explains why something is dangerous
- It supports 18 AI providers (including local models)
- It generates SBOM and checks NVD

Would love feedback from the community!
```
