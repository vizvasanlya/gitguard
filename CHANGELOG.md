### Added
- Secret detection (100+ patterns)
- Vulnerability scanning (49 patterns)
- 12 language support (Python, JS, TS, Go, Rust, Java, Ruby, PHP, C/C++, Shell, Docker, K8s)
- Auto-fix for 11 issue types
- AI-powered code review (18 providers)
- Git history scanning
- SARIF output for GitHub Security tab
- SBOM generation (CycloneDX/SPDX)
- NVD/CVE integration
- Web dashboard
- Slack/Discord notifications
- Custom rule engine (JSON/YAML)
- Git hooks management
- Entropy-based secret detection
- VS Code extension
- Homebrew formula
- GitHub Action
- 15 CLI commands
