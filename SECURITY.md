# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within GitGuard, please send an email to security@gitguard.dev. All security vulnerabilities will be promptly addressed.

## Security Updates

Security updates will be released as soon as possible after a vulnerability is confirmed.

## Scope

This security policy applies to:
- The GitGuard CLI tool
- The GitGuard Python package
- The VS Code extension
- The GitHub Action

## Out of Scope

- Third-party AI provider APIs
- Custom rules created by users
- Projects scanned by GitGuard
